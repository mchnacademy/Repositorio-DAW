EJEMPLO LISTA DE DEFINICIÓN PARA LENGUAJE DE MARCAS

SGM (Sistema de Gestión de Marca)

Es el conjunto de normas, herramientas y procesos que permiten crear, interpretar y
gestionar documentos estructurados mediante etiquetas.


SGML (Standart Generalized Markup Language)

Es un lenguaje de marcado estandarizado para definir otros lenguajes de marcado como:
HTML y XML. Proprciona una forma de estructurar datos para describir la información.


HTML (Hypertext Markup Language)

Es un lenguaje de marcado utilizado para diseñar y estrucurar contenido en la web,
como texto, imágenes, enlaces, etc.


XML (eXtensible Markup Language)

Es un lenguaje de marcado que se utiliza para almacenar, estructurar y
transportar datos, enfocado en la simplicidad y la extensibilidad


CSS (Cascading Style Sheets)

Es un lenguaje de hojas de estilo que se utiliza para definir la apariencia y
el formato de los documentos HTML. Permite controlar aspectos como colores,
fuentes, márgenes y disposición de los elementos en una página web.
