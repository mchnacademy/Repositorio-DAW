Elementos o Etiquetas de los Formularios de HTML.

Elementos input y label

En input tipo texto definimos name que es el nombre de la variable que usamos cuando enviamos el formulario, y el id que es el atributo que identifica este input y luego podemos usar en la etiqueta label.
En label usamos el atributo for para indicar a que input pertenece.

Elemento fieldset

Permite agrupar un conjunto de campos de un formulario de forma visual mediante un marco y de forma funcional permitiendo realizar acciones sobre todos los elementos agrupados. Podemos usar la etiqueta "legend" para poner un texto de título al marco.
