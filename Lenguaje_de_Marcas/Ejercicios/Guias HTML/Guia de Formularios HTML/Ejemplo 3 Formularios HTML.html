Ejemplo 3
Todos los atributos de input

Atributo value

Valor por defecto, pero puedo cambiarlo.

Atributo readonly

Solo lectura no puedo cambiarlo por que es de solo lectura, pero sienvio su valor.

Atributo disabled

No puedo cambiarlo y NO ENVIA su valor. Prueba a enviar,¿que datos se envian?

Atributo size

Es el numero de caracteres que pueden mostrarse en la caja sin desplazarse, en micaso 5 caracteres prueba con letras y luego con numeros, vemos que con numeros entran mas de 5.

Atributo maxlength

Es el numero maximo de caracteres que se puede escribir.

Atributo max y min para type number, fechas, tiempo, semana, mes y etc...

Define los numeros minimos y maximos que pueden tomar y no deja enviar otros. Selecciona una hora de cita entre 09:00 y 20:00, prueba a poner una fuera y dale a enviar.

Atributo multiple para tipos file.


Atributo pattern para tipos texto.


Atributo autocomplete para tipos texto.


Atributo placeholder para tipos texto.

Atributo required.


Atributo step para tipos number.

Atributo autofocus para text o number.
