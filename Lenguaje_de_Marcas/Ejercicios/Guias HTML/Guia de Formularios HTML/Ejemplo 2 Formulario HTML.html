Me obliga a poner un email correcto, de lo contrario nos saldra un error.

input type="password"

Me permite ocultar mi contraseña.

input type="radio"

Los botones de radio permiten al usuario seleccionar una opcion de un grupo de opciones.

IMPORTANTE: en este caso el name tiene que ser el mismo para todas las opciones.




input type="checkbox"

Las Casillas de Verificacion permiten al usuario seleccionar CERO o MAS opciones de un numero limitado de opciones.

IMPORTANTE: en este caso el name tiene que ser diferente para cada opcion.
