EJEMPLO LISTA DE DEFINICION PARA LENGUAJES DE MARCAS.

(Standart Generalized Markup Lenguage):

Es un lenguaje de marcado utilizado para definir otros lenguajes de marcado como: HTML o XML.Proporciona una forma de estructurar datos para describir la diferencia.

(eXtensible Markup Language):

Es un lenguaje de marcas extensible que permite definir tus propias etiquetas para estructurar, almacenar y transportar datos de forma legible para humanos y máquinas.

(HyperText Markup Language):

Es el lenguaje de marcado estándar para crear la estructura y el contenido de las páginas web, utilizando etiquetas para definir elementos como títulos.

(Cascading Style Sheets):

Es un lenguaje de hojas de estilo que trabaja junto a los lenguajes de marcado para definir la presentación visual.
