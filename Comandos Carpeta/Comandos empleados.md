# Lista de comandos empleados durante la elaboracion del proyecto.

---

## Comandos basicos:

- git clone (URL_repositorio) ---> Cómo clonar el repositorio.
- git checkout (nombre_rama)  ---> Cómo cambiar de rama.
- git log  ---> Cómo ver el historial de commits.
- git add (nombre_proyecto)  ---> Cómo añadir proyectos/archivos.
- git commit -m (Mensaje que explique los cambios) ---> Cómo crear un commit.
- git push origin (nombre_rama)  ---> Cómo subir cambios.
- git config --global user.name "Tu Nombre"  ---> Identificacion de usuario.
- git config --global user.email "correo@email.com"  ---> Identificacion de coeo electronico.
- git tag v1.0-entrega  ---> Crear version final del repo.
- git push –tags  ---> Como subir tags al repo.

---

## Comandos claves y trucos:

- git branch  ---> Para crear una rama.
- git reset (nombre_archivo)  ---> 
- git checkout -b (nombre_rama)  ---> Para crear una rama y situarse en ella.
- git checkout main  ---> Volver a la rama main
- git checkout (nombre-rama)  ---> Para posicionarse en una rama.
- git merge (nombre_rama)  ---> Para fusionar cambios de dos ramas.
